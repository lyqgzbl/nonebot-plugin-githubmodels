# 安装
使用nb-cli [推荐]
```shell
nb plugin install nonebot-plugin-githubmodels
```
使用pip
```shell
pip install nonebot-plugin-githubmodels
```

# 使用
命令需要加 [NoneBot 命令前缀](https://nonebot.dev/docs/appendices/config#command-start-和-command-separator) (默认为`/`)  
使用命令`AI`/`ai`触发插件  
命令选项`-r` 重置上下文记忆  
命令选项`-i` 临时启用图片回复
